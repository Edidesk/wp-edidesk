# WordPress Custom - Edidesk

Imagem Docker customizada do WordPress com configurações de PHP otimizadas.

## Uso Rápido

```bash
docker pull bartferr/wp-edidesk:latest
docker-compose up -d
```

## Configurações

- Upload: 256MB
- Memory: 512MB
- Execution time: 300s
